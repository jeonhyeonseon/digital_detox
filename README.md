## 🔔 프로젝트 소개
디지털 사용 습관을 기록하고 조절하며, 더 균형 잡힌 일상을 만들 수 있도록 돕는 웹 서비스입니다.

- **진행 기간**: 2026.02.07 ~ 진행 중
- **인원**: 4명
- **역할**: 기획, 백엔드 개발

짧은 영상, SNS, 스마트폰 과의존으로 흐트러진 일상 속에서 사용자가 자신의 디지털 사용 패턴을 인식하고
챌린지와 기록 기능을 통해 스스로 행동 변화를 만들어갈 수 있도록 기획했습니다.

---

## 주요 기능

- **디톡스 챌린지**
  - 7일 / 14일 / 30일 챌린지 제공
  - 일별 미션 수행 및 스트릭 관리

- **포커스 타이머**
  - Pomodoro 기반 집중 시간 측정
  - 완료 횟수 및 누적 집중 시간 기록

- **디지털 다이어리**
  - 날짜별 스크린타임 및 감정 기록
  - 전날 대비 사용량 변화 확인

- **커뮤니티**
  - 후기 / 질문 / 자유 게시판 운영
  - 인기글 / 최신글 조회

- **데이터 시각화**
  - 스크린타임, 집중 시간 등 변화 흐름 제공
